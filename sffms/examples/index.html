about the examples

El dinosaurio is the shortest story ever told.

To see how the title page changes with the different options, check out the various versions of the very short story La oveja negra (anon, baen, courier, daw, nonsubmission, wotf, and no options). The courier option changes the font rather than the layout. The default font may look grainy on some systems, but should print fine.

To see running headers and other features of a short story manuscript longer than one page, check out The Country of the Blind (pdf).

To see how a novel looks when typeset nicely with the nonsubmission option, check out When the Sleeper Wakes (pdf). The smart quotes option compensates for dumb quotes in the source file.
