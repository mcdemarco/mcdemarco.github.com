Recollected

October 7, 2014 | tools | jekyll | recipes

Penflip recently announced the addition of folders. I’ve been wanting to get my recipes into single files/chapters for a while, and now I have. (It was much easier with Git than in the Penflip UI.) The results are visible at Penflip and here on the site.

Once again, I did not learn enough Liquid to post-process the recipes; instead I used Jekyll data files to store (a manually YAMLed version of) the hidden Penflip contents file in a Jekyll-friendly form that is then converted into the table of contents. I found Dale Tournemille’s example more helpful in figuring out how to do that than the Jekyll docs.

It’s hard to tell what my newest recipe is after all that rearranging; it may be naan.

Collected

August 12, 2014 | tools | jekyll | recipes

Today I learned enough about Jekyll collections to import my recipes as their raw markdown source from Penflip. I had intended to learn some Liquid to do some post-processing that would have made them even more convenient to read here, but I only learned enough to seriously question that plan. Fortunately, Penflip itself seems to be back in active development, despite my initial pessimism.

I did dust the site a bit and put up a tombstone for the late lamentable Springpad (where my recipes lived before Penflip). Springpad’s death has been blamed on the Boston start-up scene and the freemium model (among other things), but I think it drove away its core user base by frequently pivoting into occupied spaces (social, Pinterest) where we didn’t want to go, breaking functionality along the way.

Tagged

July 24, 2014 | tools | jekyll | terragen

I’ve been meaning to get categories working in the blog since way back when I converted it to jekyll, and now I’ve got it working, more or less. This post, for example, should include a link to the tools tag.

I tried out a couple of plugins that did what I wanted (jekyll-category-archive-plugin and jekyll-tagging) but not quite in the way I wanted to do it, so instead I used Charlie Park’s unpackaged plugin, with a few tweaks of my own.

To add a list of all tags with post counts, I followed advice from StackOverflow and elsewhere, replacing the site.categories variable with site.tags. You can infer the details from my fork of Left if you’re brave enough.

The hard part was getting the tags back onto my posts; most of them were displaced in the move to jekyll, though I kept a list of them. While retagging old posts, I noticed that my Terragen planet tutorial, although on the list under Terragen, was missing. I recovered it from the Wayback Machine. Many links are broken (due to time and tides, not the recovery process), but my bit is there.

Crick Tickers, Cooters, and Games

July 21, 2014 | ADN

I released Crick Ticker, a web-based stream marker manager for App.net, before leaving for Pinewoods a week and a half ago. There are a few vacation photos up at Favd, starting with the red-bellied cooter, and we also played a lot of board games, which inspired me to update my collection on BoardGameGeek.

I got back very behind on my Alpha stream. Of course all my app.net iOS apps had misplaced my stream marker, and were still terrible at loading more posts without losing my place. So I got to use Crick Ticker to put my stream marker back where it belonged–several times–and found myself wanting a bit more functionality.

Today I updated Crick Ticker just a bit to show post IDs on hover and to allow you to reset an existing marker to itself (mainly in order to also set the last read marker or to set another stream’s marker to match). The code is up at github, if anyone is interested. It bears a genetic resemblance to my code for Paste.
