Offline Dictionaries

August 22, 2004 | language | tools

There are plenty of free on-line dictionaries and thesauri out there, but the selection is limited once you go off-line. WordNet is an interesting variation on the thesaurus theme. You can get a MacOS X package from Sourceforge.

JDict, is a dict-protocol dictionary server you can run locally for your own use. The easiest way to install JDict on MacOS X is with DICTatoro. The standard interface is a local web page, but the Mac version also includes a special configuration of OmniDictionary and a nice selection of free dictionaries.
